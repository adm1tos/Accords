# Accords - маленький сайт для конвертации координат Minecraft
## Что делает
Сайт получает от пользователя координаты (например в обычном мире) и конвертирует их в координаты Незера (т.е. делит на 8 - это ведь очень трудно сделать, правда?) Это также работает наоборот: Можно указать координаты из Незера и узнать, куда приведёт портал в обычном мире.

## Это ещё не все
А еще сайт может получать координаты из специальной команды (которую можно получить, нажав в игре F3 + C). Он понимает какие именно координаты и какому миру они принадлежат! Это, получается, даже вписывать координаты не нужно! Вставил строчку и забыл!
